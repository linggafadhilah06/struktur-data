BFS menjelajahi graf LEVEL demi LEVEL — persis karena sifat FIFO queue. Node yang ditemukan duluan (level dekat) diproses duluan sebelum node yang lebih jauh. Ini menjamin jalur terpendek pada graf tanpa bobot.
bfs.py — referensi kode 
def bfs(graph, start):
  visited = set()
  queue   = Queue()
  queue.enqueue(start)
  visited.add(start)

  while not queue.isEmpty():
    node = queue.dequeue()
    print(node)  # proses

    for neighbor in graph[node]:
      if neighbor not in visited:
        visited.add(neighbor)
        queue.enqueue(neighbor)
