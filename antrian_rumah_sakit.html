Teknik kunci: Pasien dengan prioritas lebih kecil (0=kritis) dilayani lebih dulu. Jika prioritas sama → FIFO (yang datang duluan dilayani duluan). Contoh: Ani dan Dedi sama-sama kritis (prioritas 0), tetapi Ani datang duluan → dilayani duluan.
antrian_rs.py — referensi kode 
pq = BPriorityQueue(4)  # 4 level prioritas

# Pasien datang
pq.enqueue("Budi",  3)  # ringan
pq.enqueue("Ani",   0)  # kritis!
pq.enqueue("Citra", 2)  # menengah
pq.enqueue("Dedi",  0)  # kritis!
pq.enqueue("Eka",   1)  # darurat

# Urutan layanan
while not pq.isEmpty():
  print(pq.dequeue())
# → Ani, Dedi, Eka, Citra, Budi
